## 🎯 What Makes PathCraft Special

Unlike typical route planners that only optimize for time/distance, PathCraft creates journeys worth remembering:

🌟 **Scenic Intelligence** - Discovers hidden waterfalls, viewpoints, heritage sites  
⚡ **Smart Routing** - Balances scenic beauty with practical efficiency  
🛏️ **Complete Planning** - Fuel stops, overnight stays, day-wise itineraries

## 🚀 Quick Start Examples

Try these natural language requests:

- **Simple route**: `Bangalore to Goa`
- **Scenic preference**: `Plan Bangalore to Goa, prefer scenic route`
- **Specific interests**: `Chikmagalur to Hampi with heritage sites`
- **Avoid tolls**: `Mysore to Coorg, avoid toll roads`

## 🧠 AI Processing Modes

PathCraft supports both offline and cloud-enhanced AI:

### 🧠 **True Offline AI** (Ollama)

- **Local LLM Processing** - All computation happens on your machine
- **Privacy-First** - Your travel plans never leave your device
- **No Internet Required** - Works completely disconnected
- **Instant Response** - No API latency, local processing

### 🤖 **AI-Powered Planning** (External APIs)

- **Advanced LLM APIs** - Leverages powerful cloud models
- **Cloud-Enhanced Intelligence** - Sophisticated natural language understanding
- **Connected Planning** - Requires internet for AI features
- **Rich Descriptions** - Detailed travel narratives and insights
